About

This is a proto-project for dictionaries for a demo it only handle colors submited in html form and comma seperated.
RAL : Strawberry red, Slate grey, Anthracite grey... X11 : MidnightBlue, SlateGray, SaddleBrown...

Hosted on heroku. On github.

Functionalities and the application logic :

  • Express used to generate RESTFUL API :
    • /api/dicts
    • /api/dict/:id
    • /api/schemas
    • /api/schemas/names
  • CRUD operations of colors dictionaries
  • CRUD of each dictionary row values
  • Schema matching to x11, RAL or create new schema
  • On Add function validation of duplicates, match to the choosed schema, save to db and highlight row if there is any problem
  • Use all new schemas beside x11, RAL to standardize your inputs values to the schema choosed
  • Feedback message on user action
  • Quick styling for the app with bootstrap
  • x11, RAL is only a starting point can be removed.
  • There is still some room for improvement and testing.

Stack :

  • Mongoose.js (MongoDB): database and mLab for heroku
  • Express.js: backend framework
  • Angular 4: frontend framework
  • TypeScript
  • Node.js: runtime environment
  • Angular CLI: project scaffolding
  • Bootstrap: layout and styles
  • Font Awesome: icons
